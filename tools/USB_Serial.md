# USB/Serial converter
## адаптер [[USB]]/[[UART]]

При работе с микроконтроллерами нужно обязательно имет как минимум один конвертер **USB/Serial**.

Одним концом втыкается в USB-порт (обычно [[USB-A]])
С другой стороны есть несколько пинов для подключения [[TTL UART]]

- [[YP-05]]
- [[HEX FT232R Breakout]]

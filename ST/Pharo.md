# Pharo

https://get.pharo.org/

```
curl https://get.pharo.org/ | bash
```

## install

```
git clone -o bb git@bitbucket.org:ponyatov/st.git ~/ST
cd ~/ST
make install
```

## run

```
cd ~/ST
make gui
```
